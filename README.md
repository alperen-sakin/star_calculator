# Star Calculator for Scrap Clicker 2

This app is inspired by the following web-based calculators:
- [jugh3ad's StarCalc](https://github.com/jugh3ad/StarCalc)
- [Riza Emet's StarCalc](https://rizaemet.github.io/StarCalc/) (Turkish version)

**Star Calculator** helps you calculate the resources required to reach a specific star level in Scrap Clicker 2. The app supports both **Turkish** and **English** languages.

### Features:
- Calculate resource costs for:
    - Star Cost (Achievement)
    - Star Cost (Master)
- Input fields for:
    - Current Star Level
    - Current Scrapyard V2 Level
    - Target Star Level
